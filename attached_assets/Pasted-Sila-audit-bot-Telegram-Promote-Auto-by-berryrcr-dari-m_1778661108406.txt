**ERROR YANG PERLU DIKESAN:**

❌ **Environment Issues:**
- BOT_TOKEN missing/invalid
- SUPABASE_URL/KEY missing
- API_ID/API_HASH missing
- DATABASE_URL error

❌ **Database Issues:**
- Table tidak wujud
- Kolum sessions incomplete (tg_username, connected_at, userbot_id, backup_email)
- RPC function tidak wujud (add_coins, deduct_coins)
- Foreign key constraint error

❌ **Telethon Issues:**
- Phone number format salah
- OTP expired/invalid
- 2FA password salah
- Session string corrupt
- Client disconnect tanpa sebab
- Flood wait error
- API rate limit exceeded
- Phone number banned

❌ **Handler Issues:**
- Double trigger (handler dipanggil 2x)
- Callback tidak answer()
- FSM state confusion
- Message duplication
- Keyboard tidak muncul

❌ **Scheduler Issues:**
- Job tidak start
- Job duplicate
- Interval tidak tepat
- Crash semasa promote
- Memory leak

❌ **Logic Issues:**
- Syiling tidak deduct
- Syiling tidak credit
- Subscription tidak active
- Group tidak tersimpan
- Mesej tidak dihantar
- Session tidak reconnect
